# Skills

NOCFoundry can generate agent-facing workflow bundles with `skills-generate`.

The current model emphasizes:

- one skill per explicit toolset
- workflow-oriented docs
- optional prompt guidance
- direct `nocfoundry invoke ...` examples instead of generated runtime wrappers
