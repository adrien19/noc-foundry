# Tools

Tool docs describe deterministic operational capabilities exposed by NOCFoundry.

## Network tools

Vendor-agnostic tools for device access and fleet operations:

- [`network-show`]({{< relref "network-show.md" >}}) — ad-hoc CLI command execution
- [`network-show-interfaces`]({{< relref "network-show-interfaces.md" >}}) — interface status and counters
- [`network-show-version`]({{< relref "network-show-version.md" >}}) — system version information
- [`network-query`]({{< relref "network-query.md" >}}) — generic profile-driven operation executor
- [`network-list-devices`]({{< relref "network-list-devices.md" >}}) — list devices with labels and transports

## Validation tools

- [`validate`]({{< relref "validate.md" >}}) — multi-step validation workflows
