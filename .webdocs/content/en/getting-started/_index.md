# Getting Started

NOCFoundry supports two common entry points:

- a local operator path for building and running the server with example configs
- an MCP client path for connecting agent tools to the protected `/mcp` surface

Use these pages first:

- [Local quickstart]({{< relref "getting-started/local-quickstart.md" >}})
- [MCP client quickstart]({{< relref "getting-started/mcp-client-quickstart.md" >}})
